# 学习方法体系应用场景

## 📊 各模块的应用场景

### 1. 财报速读器 → 投资决策
```
使用场景:
• 选股前快速扫描公司基本面
• 季度财报发布后第一时间评估
• 对比同行业公司财务指标

输出: A/B/C/D 评级 + 一页纸摘要
```

### 2. 技术分析 → 择时交易
```
使用场景:
• 买入前确认技术形态 (早晨之星/金叉)
• 持仓中监控趋势 (多头排列)
• 卖出前识别背离信号

输出: 买入/卖出/持有 建议
```

### 3. 市场监控 (akshare) → 情绪判断
```
使用场景:
• 开盘前看资金流向 (主力进还是出)
• 盘中监控龙虎榜 (游资动向)
• 盘后复盘北向资金 (外资态度)

输出: 市场情绪报告
```

### 4. 产业链分析 → 行业选股
```
使用场景:
• 行业景气度上升时选上游 (如 AI  boom 选 GPU)
• 产能过剩预警时避开中游
• 寻找国产替代机会

输出: 产业链投资地图
```

## 🎯 完整投资决策流程

```
早上 8:00
├─ FRED宏观数据 → 判断大环境
├─ 市场监控报告 → 判断情绪
└─ 全球行情简报 → 判断趋势

盘中 实时
├─ Twitter监控 → 突发事件
├─ 资金流向 → 主力动向
└─ 龙虎榜 → 游资热点

盘后复盘
├─ 个股财报 → 基本面评分
├─ 技术形态 → 买卖点确认
└─ 产业链对比 → 行业选择

定期(季度)
├─ 分红数据 → 长期持有参考
├─ 机构持仓 → 聪明钱动向
└─ 股东增减持 → 内部人信号
```

## 💡 实际使用示例

### 例1: 想买宁德时代
1. **财报速读** → 看盈利能力 (毛利率、ROE)
2. **技术形态** → 看是否底部反转
3. **资金流向** → 看主力是否在买
4. **产业链** → 看电池产业链位置

### 例2: 发现英诺赛科异动
1. **龙虎榜** → 看谁在买 (游资还是机构)
2. **资金流向** → 看持续性
3. **Twitter监控** → 看有没有消息催化
4. **技术形态** → 看是不是突破

### 例3: 季度调仓
1. **财报对比** → 同行业内选最好的
2. **分红历史** → 选回报稳定的
3. **机构持仓** → 跟随聪明钱
4. **股东减持** → 避开高管卖出的
