#### 关键概念记录

**支撑与压力**
- 支撑位：价格下跌时反复止跌的价位（买盘集中）
- 压力位：价格上涨时反复遇阻的价位（卖盘集中）
- 突破后角色互换：支撑变压力，压力变支撑

**MACD详解**
- DIF（快线）= 12日EMA - 26日EMA
- DEA（慢线）= DIF的9日EMA
- 柱状线 = DIF - DEA
- 金叉：DIF上穿DEA（买入信号）
- 死叉：DIF下穿DEA（卖出信号）
- 背离：价格新高但MACD未新高（顶部预警）

**RSI详解**
- 计算：N日内上涨幅度平均 / (上涨+下跌幅度平均) × 100
- 超买(>70)：可能回调
- 超卖(<30)：可能反弹
- 背离：价格新低但RSI未新低（底部信号）
