## 🎯 核心问题

**现状**: 简单罗列新闻和股价，缺乏深度关联  
**目标**: 建立"预期差"分析，提供可操作的结论

---

## 📊 新分析框架

### 1. 三层影响分析

```
事件 → 第一层（直接影响） → 第二层（竞争格局） → 第三层（估值重构）

例：纳微10kW技术突破
→ 第一层：股价+3.72%，但成交量？是否放量？
→ 第二层：英诺赛科需跟进，否则800V市场被抢占
→ 第三层：GaN行业门槛提升，中小玩家出局，头部集中
```

### 2. 预期差评估模型

| 维度 | 评估标准 | 输出 |
|------|----------|------|
| **事件强度** | 技术领先度、客户等级、量产进度 | 量化评分 |
| **市场反应** | 涨跌幅 vs 历史同类事件 | 超预期/符合/不及预期 |
| **资金流向** | 成交量、板块轮动、南向资金 | 资金态度 |

### 3. 对比分析框架

**每次必含**:
- 同行业公司对比（纳微vs英诺赛科估值差）
- 历史类似事件对比（谷歌合作 vs 英伟达合作）
- 技术指标验证（突破是否放量？MACD是否金叉？）

## 💡 案例：纳微10kW技术突破分析

### 表面
- 股价：+3.72%
- 新闻：发布10kW 98.5%效率平台

### 深度
- **涨幅评估**：+3.72%是否匹配技术突破？对比历史：纳微重大技术发布平均涨5-8% → **不及预期**
- **原因分析**：
  1. 市场已部分price in（APEC预告）
  2. 量产时间表不确定（2026年3月发布≠立即可买）
  3. 竞品英诺赛科刚宣布谷歌批量出货，分散注意力
- **资金态度**：成交量？需查

### 对英诺赛科影响
- **短期**：纳微技术领先，压力增大
- **中期**：若纳微量产延迟，英诺赛科窗口期
- **长期**：800V AI电源标准之争，英诺赛科谷歌背书是优势

### 操作建议
- **持有**：英诺赛科谷歌合作更落地
- **观察**：纳微3月APEC后订单公告
- **信号**：若纳微无订单跟进，资金可能回流英诺赛科
