# 📝 爆款内容知识库架构

## 核心思路
**把投资分析的框架迁移到内容创作**
- 投资：研究公司财报 → 发现价值规律
- 内容：研究爆款文章 → 发现传播规律

## 三层架构

### 第一层：知识库层 (爆款数据库)

```
content_db/
├── raw_articles/              # 原始文章存档
│   ├── x_finance/            # X平台财经类
│   │   ├── 2024_01/
│   │   ├── 2024_02/
│   │   └── ...
│   ├── x_tech/               # X平台科技类
│   └── wechat/               # 微信公众号
│
├── analyzed/                  # 已分析的文章
│   ├── title_patterns/       # 标题模式库
│   │   ├── question_type.json    # 提问式
│   │   ├── number_type.json      # 数字式
│   │   ├── contrast_type.json    # 对比式
│   │   └── emotion_type.json     # 情绪式
│   │
│   ├── hook_patterns/        # 开头钩子库
│   │   ├── story_hook.json       # 故事钩子
│   │   ├── data_hook.json        # 数据钩子

│   ├── body_patterns/        # 正文结构库
```

### 第二层：Skills层 (分析引擎)

**Skill 1: 爆款标题分析器**
- 输入: 标题文本
- 分析: 结构类型、情绪强度、关键词、长度优化
- 输出: 标题评分 + 改进建议

**Skill 2: 文章结构解析器**
- 输入: 全文
- 分析: 开头钩子、段落节奏、论证逻辑、结尾设计
- 输出: 结构拆解 + 可复用框架

**Skill 3: 情绪共鸣检测器**
- 输入: 文章内容
- 分析: 情绪曲线、共鸣点、触发分享的心理因素
- 输出: 情绪地图 + 优化建议

**Skill 4: 竞品监控器**
- 输入: 平台文章流
- 分析: 实时爆款识别、趋势追踪、差异化机会
- 输出: 热点报告 + 创作灵感

### 第三层：CRON层 (自动化)

| 时间 | 任务 | 输出 |
|------|------|------|
| 每天 9:00 | 监控昨日爆款 | 10篇爆款文章分析报告 |
| 每天 14:00 | 热点追踪 | 当日热点话题+创作建议 |
| 每周一 | 周报生成 | 上周爆款趋势+公式更新 |
| 每月1日 | 月报生成 | 爆款公式迭代+新趋势 |

## 爆款分析维度

### 1. 标题分析 (权重30%)
```
结构类型:
- 提问式: "为什么XXX?" "如何XXX?"
- 数字式: "5个方法" "3年经验"
- 对比式: "A vs B" "从X到Y"
- 情绪式: "震惊!" "泪目!" "太真实了"
- 悬念式: "没想到..." "原来..."

关键词分析:
- 高频词提取
- 情绪词强度
- 领域标签

长度优化:
- 最佳字数范围
- 移动端显示优化
```

### 2. 开头分析 (权重25%)
```
钩子类型:
- 故事钩子: 个人经历开场
- 数据钩子: 惊人数字开场
- 痛点钩子: 直击读者痛点
- 金句钩子: 名言/金句开场
- 冲突钩子: 制造悬念/冲突

首段节奏:
- 字数控制
- 信息密度
- 情绪铺垫
```

### 3. 正文分析 (权重25%)
```
结构模式:
- 问题-解决式
- 清单式 (1,2,3...)
- 故事弧线式
- 对比分析式
- 递进论证式

段落设计:
- 段落长度分布
- 小标题使用
- 金句分布
- 数据/案例插入点
```

### 4. 结尾分析 (权重20%)
```
结尾类型:
- CTA式: 引导行动
- 提问式: 开放式讨论
- 总结式: 升华主题
- 情绪式: 引发共鸣
- 悬念式: 预告下期

互动设计:
- 评论引导
- 分享触发
- 关注转化
```

## 爆款公式提炼

### 标题公式
```
公式1: [数字] + [痛点] + [解决方案]
例: "3年亏掉100万后，我总结出的5条投资铁律"

公式2: [身份标签] + [反常识观点]
例: "基金经理私下告诉我：散户永远别碰这3类股票"

公式3: [情绪词] + [具体场景]
例: "凌晨3点，我终于理解了什么是真正的财务自由"

公式4: [提问] + [悬念]
例: "为什么你越努力越穷？答案藏在这个被忽视的数据里"
```

### 结构公式
```
开场: 钩子(2-3句) → 痛点共鸣(1段) → 预告价值(1句)
正文: 观点(1句) → 论证(故事/数据/案例) → 金句升华(1句)
      (重复3-5次，形成节奏)
结尾: 总结(1段) → 情绪共鸣(1句) → CTA(1句)
```
