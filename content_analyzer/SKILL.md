## 🎯 核心用途

- **爆款分析**: 拆解高传播内容的标题、结构、情绪设计

## 🔧 核心功能

### 1. 标题分析器

**功能**: 分析标题的传播潜力

**输入**: 标题文本

**分析维度**:
- 结构类型 (提问式/数字式/对比式/情绪式/悬念式)
- 情绪强度 (0-100分)
- 关键词分析
- 长度优化建议

**输出示例**:
```json
{
  "title": "为什么90%的人投资都亏钱？",
  "score": 85,
  "type": "提问式",
  "emotion": "好奇+焦虑",
  "keywords": ["90%", "投资", "亏钱"],
  "suggestion": "数字+痛点，传播力强"
}
```

### 2. 文章结构解析器

**功能**: 拆解文章的结构框架

**输入**: 全文内容

**分析维度**:
- 开头钩子 (故事/数据/金句/痛点)
- 段落节奏 (起承转合)
- 论证逻辑
- 结尾设计 (CTA/问题/总结)

**输出示例**:
```
开头: 故事钩子 (个人亏损经历)
结构: 问题-分析-解决
节奏: 短句开头 → 长段论证 → 金句结尾
结尾: 开放式问题引导评论
```

### 3. 情绪共鸣检测器

**功能**: 分析内容的情绪曲线和共鸣点

**分析维度**:
- 情绪曲线 (焦虑→希望→行动)
- 共鸣点识别
- 分享触发因素
- 情绪地图

### 4. 竞品监控器

**功能**: 实时监控平台爆款内容

**监控平台**:
- X (Twitter) - 财经/科技类
- 微信公众号
- 知乎
- 即刻

**输出**: 热点报告 + 创作灵感

## 📊 爆款分析维度

### 标题分析 (权重30%)
| 维度 | 说明 | 示例 |
|------|------|------|
| 提问式 | 引发好奇 | "为什么XXX?" |
| 数字式 | 具体可信 | "5个方法" |
| 对比式 | 制造冲突 | "A vs B" |
| 情绪式 | 激发情感 | "震惊!" |
| 悬念式 | 留下钩子 | "没想到..." |

### 结构分析 (权重25%)
- **黄金圈结构**: Why → How → What
- **问题-解决**: 痛点 → 方案 → 效果
- **故事弧线**: 起 → 承 → 转 → 合
- **清单式**: 3/5/7个要点

### 情绪分析 (权重20%)
- 好奇 (Curiosity)
- 恐惧 (Fear)
- 希望 (Hope)
- 愤怒 (Anger)
- 归属感 (Belonging)

### 传播因子 (权重25%)
- 实用性 (Practical)
- 时效性 (Timely)
- 争议性 (Controversial)
- 情感共鸣 (Emotional)
- 身份认同 (Identity)

## 🚀 快速开始

### 分析单篇文章

```python
from content_analyzer import ContentAnalyzer

analyzer = ContentAnalyzer()

# 分析标题
title_analysis = analyzer.analyze_title("为什么90%的人投资都亏钱？")
print(f"标题评分: {title_analysis['score']}/100")

# 分析全文
article_analysis = analyzer.analyze_article(article_text)
print(f"结构类型: {article_analysis['structure']}")

```

## 📈 数据看板

### 爆款公式库

**标题公式**:
```
[数字] + [痛点] + [解决方案]
例: "5个方法，帮你告别投资焦虑"

[悬念] + [具体数字]
例: "没想到，90%的人都在犯这个错误"

[对比] + [情绪]
例: "从负债10万到财务自由，我只做了这3件事"
```

**结构公式**:
```
钩子开头 → 3个故事 → 金句结尾
痛点引入 → 原理分析 → 行动清单
```

## 🔍 故障排除

### 常见问题

**Q: 分析结果不准确？**
- 检查输入文本是否完整
- 确认文本语言支持 (目前支持中文/英文)

**Q: 监控不到新内容？**
- 检查账号列表是否更新
- 确认API配额未用完

**Q: 如何添加新的分析维度？**
- 在 `skills/` 目录创建新的分析器
- 继承 `BaseAnalyzer` 类
- 注册到 `analyzer_registry.py`
