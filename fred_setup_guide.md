# FRED API Key 快速申请指南

## 🚀 2分钟完成申请

### 步骤1: 访问网站
打开: https://fred.stlouisfed.org/

### 步骤2: 注册账号
点击右上角 **"My Account"** → **"Register"**

填写信息:
- First Name: [你的名字]
- Last Name: [你的姓氏]
- Email: [你的邮箱]
- Password: [设置密码]
- Organization: Individual (个人)
- Country: China

### 步骤3: 验证邮箱
查收邮件，点击验证链接

### 步骤4: 申请API Key
1. 登录后点击 **"My Account"** → **"API Keys"**
2. 点击 **"Request API Key"**
3. 填写用途: "Personal investment research"
4. 提交后**立即获得** Key

### 步骤5: 提供给我
把获得的Key发给我，格式:
```
abcdefghijklmnopqrstuvwxyz123456
```
