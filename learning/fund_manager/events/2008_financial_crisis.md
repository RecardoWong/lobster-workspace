# 📉 2008 金融危机复盘

## 📅 时间线

### 2007年: 危机酝酿
- **2月**: HSBC宣布次级贷损失106亿美元
- **4月**: 新世纪金融公司破产 (美国第二大次级贷机构)
- **6月**: 贝尔斯登两只对冲基金倒闭
- **8月**: BNP Paribas冻结三只基金，全球信贷市场冻结

### 2008年: 危机爆发
- **3月**: 贝尔斯登被摩根大通收购 ($2/股，原价$170)
- **9月7日**: 美国政府接管房利美、房地美
- **9月15日**: **雷曼兄弟破产** (史上最大破产案，$6390亿资产)
- **9月16日**: AIG被政府接管 ($850亿救助)
- **9月19日**: 货币市场基金恐慌，政府提供担保
- **10月**: 全球股市崩盘
  - 标普500 10月单月 -16.9%
  - VIX 峰值 89.53 (历史最高)

### 2009年: 底部与复苏
- **3月6日**: 标普500触及666点 (底部)
- **3月**: QE1启动，美联储购买$3000亿国债

## 💡 关键教训

### 1. 风险传染路径
```
次级贷违约 → 房价下跌 → MBS/CDO贬值 → 金融机构亏损 → 信贷冻结 → 实体经济衰退
```

### 2. 杠杆的杀伤力
- 投行杠杆率高达30-40倍
- 影子银行系统缺乏监管

### 3. 政策应对
- **货币政策**: 降息至0% + QE
- **财政政策**: TARP ($7000亿救助计划)
- **非常规工具**: 直接购买企业债、救助AIG

### 4. 资产表现
| 资产 | 2008年表现 | 2009-2019表现 |
|------|-----------|--------------|
| 标普500 | -38.5% | +250% |
| 纳斯达克 | -40.5% | +400% |
| 黄金 | +5.8% | +100% |
| 美债 | +20% | +30% |
| 房地产 | -30% | +50% |

### 5. 可复用的信号
- 信贷利差扩大 (TED Spread, Libor-OIS)
- 金融机构股价暴跌
- 货币市场基金流出
- VIX 突破30为警告，突破50为危机

## 📊 复盘要点

### 为什么没能预测？
1. **复杂性**: 衍生品链条太长，没人知道风险在哪
2. **评级误导**: AAA评级的MBS其实是垃圾
3. **过度自信**: "这次不一样"、"房价永远不会跌"

### 如何应对？
1. **识别信号**: 信贷市场冻结是早期警告
2. **仓位管理**: 保留现金，降低杠杆
3. **逆向投资**: 2009年3月是历史性买入机会
