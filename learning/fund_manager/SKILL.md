# Skill: 基金经理培训

## 描述
基金经理能力培养系统，包括宏观经济分析、个股财报研究和市场事件复盘。

## 用途
- 学习美联储政策与宏观数据解读
- 分析美股Top 50公司财报
- 复盘重大市场事件（2008金融危机、2020疫情等）
- 生成每日投资扫描报告

## 触发条件
用户提及以下关键词时激活：
- "基金经理"
- "投资学习"
- "财报分析"
- "宏观经济"
- "市场复盘"
- "每日扫描"

## 执行流程

### 1. 每日市场扫描（自动）✅ 已实现
```
每天早上6:30自动执行：
- 获取财经新闻情感分析 ✅ NewsAPI + VADER
- 扫描市场热点和板块轮动 ✅ 从新闻中提取
- 生成每日市场摘要报告 ✅ 自动生成
```

**状态**: 新闻情感分析已实现 (NewsAPI + VADER完全免费)
**待接入**: FRED宏观数据 (已有模块，待整合)

### 2. 财报速读（按需）✅ 已实现
```
用户提供股票代码 → 
分析最新财报 → 
输出：营收/利润/指引 + 健康度评分
```

**支持市场**:
- ✅ 美股: yfinance (免费)
- ✅ 港股: AkShare (免费)
- ✅ A股: AkShare (免费)

**评分维度**: 5维健康度评分
- 成长性 (30%): 营收/利润增速
- 盈利能力 (25%): 毛利率/ROE/净利率
- 运营效率 (20%): 现金流/周转
- 财务安全 (15%): 负债率/流动比率
- 估值水平 (10%): PE/PB

### 3. 事件复盘（按需）⏳ 待实现
```
用户指定事件（如"2008金融危机"）→
调取历史数据 →
分析起因/过程/结果/教训
```
