# 📈 美联储利率决议历史 (2014-2024)

## 背景知识
- **联邦基金利率**: 银行间隔夜拆借利率，美联储通过此利率调控货币政策
- **加息周期**: 抑制通胀，经济过热
- **降息周期**: 刺激经济，应对衰退

## 关键时间节点

### 2014-2015: 退出QE，首次加息
- 2014年10月: QE正式结束
- 2015年12月: 首次加息 0.25% (0->0.25%->0.5%)
  - 背景: 失业率降至5%，经济企稳
  - 市场反应: 标普500先跌后涨

### 2016-2018: 缓慢加息周期
- 2016年12月: +25bp -> 0.75%
- 2017年3月: +25bp -> 1.0%
- 2017年6月: +25bp -> 1.25%
- 2017年12月: +25bp -> 1.5%
- 2018年3月: +25bp -> 1.75%
- 2018年6月: +25bp -> 2.0%
- 2018年9月: +25bp -> 2.25%
- 2018年12月: +25bp -> 2.5%
  - 背景: 经济强劲，通胀接近2%目标
  - 2018年12月最后一次加息后市场大跌，鲍威尔态度转鸽

### 2019: 预防性降息
- 2019年7月: -25bp -> 2.25% (预防性降息)
- 2019年9月: -25bp -> 2.0%
- 2019年10月: -25bp -> 1.75%
  - 背景: 贸易紧张、全球经济放缓担忧
  - 被称为"保险性降息"

### 2020: COVID-19 紧急降息
- 2020年3月3日: -50bp -> 1.25% (紧急降息)
- 2020年3月15日: -100bp -> 0.25% (紧急降息至零利率)
  - 背景: 疫情爆发，金融市场恐慌
  - 市场反应: 短暂反弹后继续暴跌，直到3月23日触底

### 2020-2021: 零利率 + QE
- 维持0-0.25%长达2年
- 无限制QE购买美债和MBS
- 通胀开始抬头但联储认为是"暂时的"

### 2022: 激进加息周期 (40年来最快)
- 2022年3月: +25bp -> 0.5% (首次加息)
- 2022年5月: +50bp -> 1.0%
- 2022年6月: +75bp -> 1.75% (首次75bp)
- 2022年7月: +75bp -> 2.5%
- 2022年9月: +75bp -> 3.25%
- 2022年11月: +75bp -> 4.0%
- 2022年12月: +50bp -> 4.5%
  - 背景: 通胀高达9.1% (1981年来最高)
  - 市场反应: 全年科技股暴跌，NASDAQ -33%

### 2023: 加息放缓但持续
- 2023年2月: +25bp -> 4.75%
- 2023年3月: +25bp -> 5.0%
  - 背景: 硅谷银行SVB倒闭，区域性银行危机
  - 联储一边加息一边提供流动性工具
- 2023年5月: +25bp -> 5.25%
- 2023年7月: +25bp -> 5.5% (本轮周期最高点)
  - 之后进入"更高更久"(higher for longer)阶段

### 2024: 降息周期开启
- 2024年9月: -50bp -> 5.0% (首次降息，超预期)
- 2024年11月: -25bp -> 4.75%
- 2024年12月: -25bp -> 4.5%

## 📊 关键洞察

### 1. 利率与股市关系
- **短期**: 加息通常利空股市，降息利好
- **长期**: 取决于经济基本面
- **例外**: 2023年加息但AI推动科技股大涨

### 2. 利率与债券
- 利率↑ = 债券价格↓
- 2022年美债历史性熊市

### 3. 利率与美元
- 加息周期美元走强 (2018, 2022)
- 降息周期美元走弱

### 4. 市场预测 vs 实际
- 点阵图 (Dot Plot) 经常不准
- 市场往往领先或落后联储行动
