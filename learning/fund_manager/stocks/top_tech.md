# 🏢 美股 Top 50 公司分析框架

## 科技板块 (Magnificent 7)

### Apple (AAPL)
**商业模式**: 硬件 + 软件 + 服务生态
**关键指标**:
- iPhone 收入占比 (~50%)
- 服务收入增速 (高毛利业务)
- 毛利率 (目标45%以上)
- 自由现金流 (行业顶尖)

**投资主题**:
- 服务业务转型
- Vision Pro 新品类
- 印度等新兴市场
- 股票回购 (每年$900亿)

---

### Microsoft (MSFT)
**商业模式**: 云计算 + 企业软件 + 个人计算
**关键指标**:
- Azure 增长率 (目标40%+)
- 云收入占比
- Office 365 订阅数
- 毛利率 (超70%)

**投资主题**:
- AI + Azure (OpenAI合作)
- Copilot 产品矩阵
- 企业软件护城河

---

### NVIDIA (NVDA)
**商业模式**: GPU + AI芯片 + 数据中心
**关键指标**:
- 数据中心收入增速
- 毛利率 (超70%)
- 新品发布周期
- 竞争对手份额 (AMD, Intel)

**投资主题**:
- AI训练/推理需求爆发
- 数据中心芯片垄断地位
- 汽车/机器人新增长曲线
- 风险: 估值过高、地缘政治

---

### Alphabet/Google (GOOGL)
**商业模式**: 搜索广告 + YouTube + 云 + Waymo
**关键指标**:
- 搜索广告收入
- YouTube 收入
- Google Cloud 增速
- 运营利润率 (目标30%+)

**投资主题**:
- AI搜索竞争 (vs Microsoft)

- Waymo 自动驾驶估值

---

## 分析模板

### 季度财报检查清单
```
□ 营收 growth YoY/QoQ
□ EPS beat/miss vs 预期
□ 毛利率趋势
□ 运营利润率趋势
□ FCF 状况
□ 指引: 上调/下调/维持
□ 各业务线表现
□ 地域收入分布
□ 风险因素更新
```

### 估值框架
```
当前PE: X倍
历史PE区间: Y-Z倍
PEG比率: X (增长调整)
EV/EBITDA: X倍
与同行对比: 溢价/折价
DCF估值: $X (假设增长率Y%)
```

### 催化剂/风险
```
催化剂:
- 新产品发布
- 市场份额提升
- 利润率改善
- 股票回购增加

风险:
- 竞争加剧
- 监管压力
- 宏观放缓
- 估值压缩
```
