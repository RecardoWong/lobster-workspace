## 📝 笔记模板

### 宏观数据笔记
```
日期: YYYY-MM-DD
事件: [利率决议/CPI公布/非农等]
数据: 实际值 vs 预期值 vs 前值
市场反应: 
- 标普500: +/- X%
- 纳斯达克: +/- X%
- 10年美债: +/- Xbp
- 美元指数: +/- X%
逻辑分析:
- 为什么市场这样反应？
- 与历史对比如何？
- 对后续影响？
```

### 财报分析笔记
```
公司: [代码]
季度: QX YYYY
关键指标:
- 营收: $X B (预期 $Y B) ->  beat/miss
- EPS: $X (预期 $Y) -> beat/miss
- 指引: 上调/下调/维持
- 毛利率: X%
- 自由现金流: $X B
业务亮点:
- 各业务线增长情况
- 地域分布
- 新订单/产品
投资者关注点:
- AI/云业务进展
- 成本控制
- 竞争格局
盘后反应: +/- X%
```

### 事件复盘笔记
```
事件: [名称]
时间: YYYY-MM-DD ~ YYYY-MM-DD
背景:
- 当时宏观环境
- 市场估值水平
- 关键催化剂
演变过程:
- Phase 1: 初期反应
- Phase 2: 恐慌/贪婪峰值
- Phase 3: 政策应对
- Phase 4: 恢复期
资产表现:
- 标普500: 高点->低点->恢复 (跌幅X%, 恢复Y天)
- VIX: 峰值 X
- 避险资产: 黄金/美债/美元表现
- 板块轮动: 哪些行业抗跌/领涨
经验教训:
- 信号识别
- 应对策略
- 可复用的框架
```

## 📅 学习计划
- 每周学习 2-3 个宏观数据点
- 每月深度分析 5 家公司财报
- 每季度复盘 1 个重大事件
