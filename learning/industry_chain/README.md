# 产业链分析系统 (Industry Chain)

## 学习目标
梳理产业链上下游关系，识别关键环节和投资机会。

---

## 📊 模块架构

### 1. 产业链图谱构建器
**功能**：绘制行业上下游关系图

**支持产业链**：
- 🏭 AI数据中心：GPU → 服务器 → 数据中心 → 云服务
- 🔋 新能源车：锂矿 → 电池 → 整车 → 充电设施
- 📱 消费电子：芯片 → 模组 → 终端 → 渠道
- 🌾 农业：种子 → 种植 → 加工 → 零售

### 2. 价值链分析器
**功能**：分析各环节利润率、议价能力

**分析维度**：
- 💰 毛利率分布（哪环最赚钱？）
- 📈 增速分布（哪环增长最快？）
- 🏆 集中度（CR5 市占率）
- 🔒 进入壁垒（技术/资金/政策）

**价值转移趋势**：
- 上游：技术突破 → 利润提升
- 中游：产能过剩 → 利润压缩
- 下游：规模效应 → 利润稳定

---

### 3. 景气度追踪器
**功能**：监控产业链各环节景气指标

**先行指标**：
- 订单/合同金额（领先1-2季度）
- 设备利用率
- 库存水平

**同步指标**：
- 营收增速
- 出货量

**滞后指标**：
- 利润率变化
- 产能扩张计划
