# Skill: 产业链分析

## 描述
产业链分析系统，梳理上下游关系，识别关键环节和投资机会。

## 用途
- 构建产业链图谱（上中下游）
- 分析价值链分布（哪环最赚钱）
- 追踪行业景气度
- 识别瓶颈环节和投资机会

## 触发条件
用户提及以下关键词时激活：
- "产业链"
- "上下游"
- "供应链"
- "价值链"
- "AI数据中心"
- "半导体"
- "新能源"

## 执行流程

### 1. 产业链图谱构建
```
输入: 行业名称（如"AI数据中心"）
输出: {
  "upstream": [{"name": "GPU/芯片", "players": ["NVDA"], "bottleneck": true}],
  "midstream": [{"name": "服务器代工", "players": ["工业富联"]}],
  "downstream": [{"name": "云服务", "players": ["AWS"]}]
}
```

### 2. 价值链分析
```
分析: 各环节毛利率、议价能力、集中度
输出: {
  "upstream_margin": "60-80%",
  "midstream_margin": "10-20%",
  "downstream_margin": "25-35%",
  "profit_concentration": "上游"
}
```

### 3. 景气度追踪
```
监控指标:
- 先行: 订单金额、设备利用率
- 同步: 营收增速、出货量
- 滞后: 利润率、产能计划
```

## 支持产业链

| 产业链 | 上游 | 中游 | 下游 |
|--------|------|------|------|
| AI数据中心 | GPU/HBM/封装 | 服务器/光模块/散热 | 云服务/企业客户 |
| 新能源车 | 锂矿/正极材料 | 电池/电机 | 整车/充电设施 |
| 半导体 | 设备/材料 | 制造/封测 | 设计/终端 |
| 光伏 | 硅料 | 硅片/电池/组件 | 电站/储能 |
