# 技术分析学习系统 (Technical Analysis)

## 学习目标
掌握K线形态识别、均线系统、量价关系等技术分析方法。

---

## 📊 模块架构

### 1. K线形态识别器
**功能**：识别常见K线形态，判断多空信号

**支持形态**：
- 单根K线：大阳线、大阴线、十字星、锤子线
- 两根K线：吞没形态、刺透形态、乌云盖顶
- 三根K线：早晨之星、黄昏之星、红三兵
- 持续形态：旗形、三角形、楔形

**输出格式**：
```json
{
  "pattern": "早晨之星",
  "type": "反转看涨",
  "confidence": 0.85,
  "suggestion": "关注后续成交量配合"
}
```

---

### 2. 均线系统分析器
**功能**：计算多条均线，识别多头排列/空头排列

**支持均线**：
- 短期：MA5, MA10
- 中期：MA20, MA60
- 长期：MA120, MA250

**信号类型**：
- 金叉/死叉
- 多头排列（MA5 > MA10 > MA20 > MA60）
- 空头排列
- 均线粘合（变盘信号）

---

### 3. 量价关系分析器
**功能**：分析成交量与价格的关系

**经典量价组合**：
- 量价齐升 → 健康上涨
- 量价背离 → 警惕反转
- 缩量回调 → 洗盘可能
- 放量滞涨 → 主力出货
