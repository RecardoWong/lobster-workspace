# Skill: 技术分析

## 描述
技术分析学习系统，包括K线形态识别、均线系统、量价关系分析。

## 用途
- 识别K线形态（单根/多根组合）
- 计算和分析均线系统
- 判断量价关系（量价齐升、背离等）
- 生成技术分析信号

## 触发条件
用户提及以下关键词时激活：
- "技术分析"
- "K线形态"
- "均线系统"
- "量价关系"
- "金叉死叉"
- "支撑位压力位"

## 执行流程

### 1. K线形态识别
```
输入: 股票代码 + 时间周期
输出: {
  "pattern": "早晨之星",
  "type": "反转看涨",
  "confidence": 0.85,
  "suggestion": "关注后续成交量配合"
}
```

### 2. 均线系统分析
```
输入: 股票代码
计算: MA5/10/20/60/120/250
输出: {
  "ma5": XX.XX,
  "ma20": XX.XX,
  "signals": ["金叉", "多头排列"],
  "trend": "上升趋势"
}
```

### 3. 量价关系分析
```
输入: 股票代码
分析: 成交量与价格关系
输出: {
  "pattern": "量价齐升",
  "signal": "健康上涨",
  "risk": "低"
}
```

## 支持形态

### 单根K线
- 大阳线、大阴线、十字星、锤子线

### 多根K线组合
- 吞没形态、刺透形态、乌云盖顶
- 早晨之星、黄昏之星、红三兵

### 持续形态
- 旗形、三角形、楔形

## 均线周期
- 短期: MA5, MA10
- 中期: MA20, MA60
- 长期: MA120, MA250
