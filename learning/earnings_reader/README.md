# 财报速读系统 (Earnings Reader)

## 学习目标
快速提取财报关键指标，5分钟内掌握公司核心财务健康状况。

---

## 📊 模块架构

### 1. 核心指标提取器
**功能**：自动提取三大报表关键数据

**利润表指标**：
- 营收 YoY / QoQ 增速
- 毛利率、净利率变化
- 费用率（研发/销售/管理）
- EPS 及超预期幅度

**资产负债表指标**：
- 现金及现金等价物
- 有息负债率
- 存货周转天数
- 应收账款周转天数

**现金流量表指标**：
- 经营现金流
- 自由现金流
- 资本支出

---

### 2. 健康度评分器
**功能**：综合打分，快速判断公司质量

**评分维度**（每项0-100分）：
```
成长性 (30%): 营收增速 + 利润增速
盈利能力 (25%): 毛利率 + ROE + ROIC
运营效率 (20%): 周转率 + 费用控制
财务安全 (15%): 现金流 + 负债率
估值水平 (10%): PE/PB/PS 分位数
```

**输出**：
- 总分（A/B/C/D 评级）
- 各维度雷达图
- 与同业对比

---

### 3. 异常预警器
**功能**：识别财报"雷区"

**常见异常信号**：
- ⚠️ 营收增长但经营现金流下降
- ⚠️ 应收账款增速 > 营收增速
- ⚠️ 存货异常增加
- ⚠️ 毛利率突然下滑
- ⚠️ 商誉减值风险
- ⚠️ 关联交易增加
