# Skill: 财报速读

## 描述
财报速读系统，5分钟内提取关键指标，判断公司财务健康状况。

## 用途
- 快速提取利润表/资产负债表/现金流量表核心指标
- 综合健康度评分（A/B/C/D评级）
- 识别财报异常信号（雷区预警）
- 生成"一页纸"速读报告

## 触发条件
用户提及以下关键词时激活：
- "财报"
- "财报速读"
- "业绩"
- "季报"
- "年报"
- "财务分析"
- "EPS"

## 执行流程

### 1. 核心指标提取
```
输入: 股票代码
获取: 最新季度财报
输出: {
  "revenue": "12.3B (+45% YoY)",
  "gross_margin": "28.5%",
  "eps": "$2.5 (beat by $0.3)",
  "operating_cash_flow": "-2.1B"
}
```

### 2. 健康度评分
```
成长性 (30%): 营收增速 + 利润增速
盈利能力 (25%): 毛利率 + ROE + ROIC
运营效率 (20%): 周转率 + 费用控制
财务安全 (15%): 现金流 + 负债率
估值水平 (10%): PE/PB/PS 分位数
```

### 3. 异常预警
```
⚠️ 营收增长但经营现金流下降
⚠️ 应收账款增速 > 营收增速
⚠️ 存货异常增加
⚠️ 毛利率突然下滑
```

## 评分维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 成长性 | 30% | 营收增速 + 利润增速 |
| 盈利能力 | 25% | 毛利率 + ROE + ROIC |
| 运营效率 | 20% | 周转率 + 费用控制 |
| 财务安全 | 15% | 现金流 + 负债率 |
| 估值水平 | 10% | PE/PB/PS 分位数 |

## 评级标准
- **A (85-100分)**: 优秀，核心持仓候选
- **B (70-84分)**: 良好，可考虑买入
- **C (60-69分)**: 一般，需进一步分析
- **D (<60分)**: 较差，建议回避
