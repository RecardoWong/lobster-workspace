# 示例设计文档: 添加实时黄金价格卡片

## 目标
在 Dashboard 左侧添加一个显示实时黄金价格的卡片

## 布局
- 位置: 上游供应商卡片下方
- 宽度: 与上游供应商卡片相同（跟随grid）
- 高度: 自适应（约150px）

## 数据结构
```json
{
  "price": 2034.50,
  "change": 12.30,
  "change_percent": 0.61,
  "currency": "USD",
  "unit": "oz",
  "update_time": "2026-02-19 23:00:00"
}
```

## 数据来源
- 来源: 新浪财经黄金API
- 更新频率: 每5分钟
- 备用: 如果新浪财经不可用，使用固定提示"数据更新中"

## 视觉设计
- 卡片背景: 金色渐变 (#ffd70020 到 #ffd70005)
- 边框: 左侧3px金色 (#ffd700)
- 价格字体: 24px 粗体
- 涨跌幅: 绿色/红色标准样式
